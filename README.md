# VGG_Clothes



## Setup

### Prerequisite
* Python 3.6
* VGG16 keras model
* tensorflow-gpu

#### Run the code
```sh
 python vgg_two.py --dataset animals \
--model output/simple_nn.model \
--label-bin output/simple_nn_lb.pickle \  
--plot output/simple_nn_plot.png

```
